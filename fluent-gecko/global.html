defaultGenerateMessages()

The default localization strategy for Gecko. It comabines locales available in L10nRegistry, with locales requested by the user to generate the iterator over MessageContexts.

In the future, we may want to allow certain modules to override this with a different negotitation strategy to allow for the module to be localized into a different language - for example DevTools.
